Project overview

FotoWare for Android/iOS is a companion app for managing your FotoWare Digital Asset Management system on the go.
The app allows you to find, preview, describe assets with metadata, run approval workflows, and upload new assets. You can also share content with teammates or external contacts quickly.

My contributions

The application was created by an external consultant before I started my position at Fotoware.

  • Adaptation of web features to mobile interface
  • Create user flows
  • Usability testing
Consent management

Feature development

Consent Management is a feature provided by Fotoware. This feature gives Fotowares customers a tool to send, receive, manage and link consent forms to media files located in their DAM. This gives the companies a clear overview of which media files can and cannot be used, or in which categories of consent have been specified.

Crop & download

Feature development

Crop & download is a feature that allows the user to download media files in set dimensions. Companies can have preset sizes for different uses or they can freely decide their own ratio.

User flow

Each feature has its own user flow, providing the team with a clear overview of necessary tasks, interactions, and data sets. I typically initiate projects with a meeting to establish the user flow, ensuring everyone understands feasibility, limitations, and project scope. This collaborative stage is crucial, involving stakeholders and project managers, as it helps us create a solid plan and prevents scope creep that could lead to rework. As the project progresses, the user flow adapts accordingly.
